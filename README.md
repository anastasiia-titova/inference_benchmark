# Inference Benchmark & Sampling Analysis
**Задача:** Разработка системы бенчмаркинга скорости инференса LLM и анализа влияния параметров сэмплинга на производительность и качество генерации.

**Технологии:** PyTorch, Transformers (Hugging Face), CUDA, Time.

**Ключевые моменты реализации:**
- Замер скорости генерации в токенах в секунду для оценки производительности модели на разном оборудовании (CPU/GPU).
- Тестирование параметров сэмплинга (temperature, top_k, top_p) для понимания их влияния на скорость и разнообразие выходного текста.
- Измерение времени полной генерации и усреднение результатов по нескольким запускам для получения стабильных метрик.
- Демонстрация работы алгоритмов Greedy Search, Top-K и Top-P (Nucleus) сэмплинга на практических примерах.
